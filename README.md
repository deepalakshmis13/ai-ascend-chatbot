# 🤖 Multilingual Government Chatbot (AI ASCEND 2026)

## 📌 Problem Statement

Accessing government schemes and public services is often difficult due to language barriers and lack of awareness. Citizens struggle to find accurate and simple information.

## 💡 Proposed Solution

We propose a **Multilingual AI Chatbot** that helps users understand government schemes and services in their preferred language (English/Tamil).

The chatbot provides:

* Instant responses
* Simple explanations
* Multilingual support

---

## ⚙️ Tech Stack (AWS)

* Amazon Lex – Chatbot development
* AWS Lambda – Backend processing
* Amazon Translate – Language translation
* Amazon S3 – Data storage

---

## 🏗️ Proposed Architecture

User → Chatbot (Lex) → Lambda → S3 → Response
↓
Translate (if needed)

---

## ✨ Key Features

* 🌐 Multilingual support (Tamil + English)
* 🤖 AI-based conversational interface
* 📄 Government scheme information
* ⚡ Fast and scalable cloud solution

---
## 👤 Use Case
A user asks:
"எனக்கு அரசு உதவி திட்டங்கள் என்ன கிடைக்கும்?"

The chatbot responds with relevant schemes in Tamil/English.
